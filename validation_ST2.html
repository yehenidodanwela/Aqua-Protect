Page Editor - Student 02 (Home Page)

Table Page validation report

I found several issues during validation, including unnecessary ARIA roles such as "navigation" on the nav element and "columnheader" on the th elements, which are redundant as these roles are already defined by the elements themselves. I also found improper use of aria-label on td elements that contain visible text. To resolve these, I removed the redundant roles and ensured aria-label was used only where necessary.
